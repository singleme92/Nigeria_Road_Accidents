## 📌 Project Overview

Road traffic accidents remain one of Nigeria's major public safety challenges, leading to significant loss of lives, injuries, and economic costs each year.

This Power BI project analyzes road accident data across Nigerian states to uncover patterns, identify accident hotspots, compare accident severity, and provide actionable insights for road safety stakeholders.

The dashboard enables decision-makers to monitor accident trends and better understand casualty distribution across the country.

---

# 🎯 Objectives

The project aims to answer the following business questions:

- Which states recorded the highest number of road accidents?
- Which states recorded the highest number of deaths?
- How are accident severities distributed?
- What proportion of casualties resulted in fatalities?
- Which states require greater road safety intervention?
- What trends can be observed across reporting periods?

---

# 📊 Dataset Information

The dataset contains road accident records for Nigerian states with the following fields:

| Column | Description |
|---------|-------------|
| STATE ID | Unique State Identifier |
| STATE | Nigerian State |
| FATAL | Fatal Accident Cases |
| SERIOUS | Serious Accident Cases |
| MINOR | Minor Accident Cases |
| TOTAL CASES | Total Recorded Accident Cases |
| NUMBER INJURED | Total Persons Injured |
| NUMBER KILLED | Total Deaths |
| TOTAL CASUALTY | Total Casualties |
| PEOPLE INVOLVED | Total People Involved |
| PERIOD | Reporting Period |

# 📌 Key Insights

- Certain states consistently report significantly higher accident cases than others.
- Fatal accidents represent a smaller proportion of total accidents but account for substantial loss of life.
- Serious accidents contribute heavily to total casualties.
- States with the highest accident volumes also tend to record higher numbers of injuries and fatalities.
- The dashboard enables quick identification of accident hotspots for road safety planning.
